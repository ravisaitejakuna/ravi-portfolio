# Ravi Portfolio (React + Tailwind + Framer Motion)

## Quick Start
```bash
npm install
npm run dev
```
Then open the URL shown in the terminal (usually http://localhost:5173).

## Build for production
```bash
npm run build
npm run preview
```

## Notes
- To change the downloaded resume, replace `public/resume.pdf` and the code uses `/resume.pdf` path.
- Toggle dark/light mode with the top-right button. The preference is saved in localStorage.
- Edit `src/App.jsx` to update content (skills, projects, etc.).
